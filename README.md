# crimson.crawler
Переобход страниц для bitrix

Парсит заданную карту сайта в отдельном потоке средствами сервера. Специально ограничен запуск только одного парсера из за большой нагрузки на сервер.

## Генерирует отчёт
* Статус страницы 
* Время загрузки
* Содержимое страницы
